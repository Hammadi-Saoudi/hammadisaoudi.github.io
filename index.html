// 01 · about me

À propos de moi

Je suis Hamadi Saoudi, étudiant en 3ème année Génie des Télécommunications à l'École Nationale d'Ingénieurs de Tunis (ENIT). Ma formation m'a donné des bases solides en traitement du signal, réseaux et systèmes embarqués, que j'ai enrichies par une passion personnelle pour l'IA.

Ce qui me motive : utiliser le Machine Learning et le Deep Learning pour résoudre des problèmes concrets — classification médicale, détection d'anomalies réseau, vision par ordinateur.

En dehors du code, je lis des articles de recherche sur arXiv, je contribue à des compétitions Kaggle, et j'explore la sécurité des systèmes télécom.

// 03 · projects

Projets

Classification de maladies neurodégénératives Deep Learning

// Pourquoi ce projet ?

Motivé par l'impact des diagnostics tardifs sur les patients atteints d'Alzheimer et Parkinson, j'ai développé un modèle CNN capable de classifier ces maladies à partir d'IRM cérébrales. Le modèle atteint 94% de précision sur le dataset ADNI — comparable à des approches publiées dans des revues médicales.

Python TensorFlow OpenCV Kaggle

Détection d'anomalies réseau par ML Cybersécurité

// Pourquoi ce projet ?

Les attaques DDoS coûtent des millions aux entreprises. J'ai conçu un système IDS basé sur Random Forest et XGBoost pour identifier en temps réel des attaques sur le dataset NSL-KDD, avec un taux de détection de 97.3% et moins de 0.5% de faux positifs.

Python Scikit-learn XGBoost Wireshark
GitHub →
Optimisation de couverture réseau 5G par IA Télécom

// Pourquoi ce projet ?

Projet académique liant ma formation télécom à l'IA : utilisation de reinforcement learning pour optimiser le placement d'antennes 5G dans un environnement urbain simulé, réduisant les zones d'ombre de 38% par rapport aux méthodes classiques.

Python MATLAB Reinforcement Learning NumPy
